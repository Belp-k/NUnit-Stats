# NUnit-Stats

## Dependencies
Developped using
 - python **3.6.4**
 - xlwt **1.3.0**

## Usage
```
> python main.py [options]

Options:
  -h, --help            show this help message and exit
  -i REPORT, --input=REPORT
                        NUnit XML report file name
  -o FILENAME, --output=FILENAME
                        Generated Excel file name

```
